NativeScript (ˈnā•tiv•skript) noun.
1. A free and open source framework for building native iOS and Android apps using JavaScript and CSS.
NativeScript renders UIs with the native platform’s rendering engine—no WebViews—resulting in native-like performance and UX.

Creating a project


                            $ tns create my-app
                            # Then after that command finishes:
                            $ cd my-app

Running the app


                            $ tns run ios --emulator
                        

                            $ tns run android

Creating a component

  • Using the nativescript-component module 

                            $ npm install nativescript-component --save

Component JS


                            // app/components/my-component/my-component.js
                            import Component from 'nativescript-component';

                            class MyComponent extends Component {

                                init() {

                                    this.set('message', 'Hello, world!');
                                }
                            }
                            MyComponent.export(exports);

Hooking up actions

Hooking up actions


                            // app/components/my-component/my-component.js
                            import Component from 'nativescript-component';

                            class MyComponent extends Component {

                                init() {

                                    this.set('message', 'Hello, world!');
                                }

                                onTap() {

                                    alert('Great job!');
                                }
                            }
                            MyComponent.export(exports);

Platform-specific code


                            // app/components/my-component/my-component.js
                            import Component from 'nativescript-component';
                            import { device } from 'tns-core-modules/platform';

                            class MyComponent extends Component {

                                init() {

                                    if (device.os.includes('iOS')) {
                                        this.set('message', 'Hello, iOS!');
                                    } else {
                                        this.set('message', 'Hello, Android!');
                                    }
                                }

                                onTap() {

                                    alert('Great job!');
                                }
                            }
                            MyComponent.export(exports);
